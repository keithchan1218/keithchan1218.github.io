## What's new

#### Arrow Function
```javascript
  (parameters)=>{
    console.log("comments");
  }
```
It is different with regular function in JavaScript,
An alternative way to address the **this** scope problem.
Many JavaScript developer should face the **this** scope issue.
We can use arrow function keep the **this** stick and point to its parent.

#### Spread Operator
```javascript
  let array = [1,2,3];
  console.log(...array); // result = 6
  
  let array = ['Tom','Jerry'];
  console.log(...array); // TomJerry
```
It is simple and useful for combine multiple arrays.
